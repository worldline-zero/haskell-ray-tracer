# A ray tracer written in haskell

A simple raycasting renderer written in haskell.

Features:
- Bounding volume hierarchies
- Wavefront.obj and Material Template Library loading

Features planned:
- Simple PBR
- Hardware acceleration via OpenGL compute shader

Currently in development.

## Install and run

```
$ git clone https://github.com/worldline-zero/haskell-ray-tracer.git
$ make
$ ./Main
```
